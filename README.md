# Análise Histórica das Seleções Campeãs da Copa do Mundo

Projeto desenvolvido para a disciplina de Banco de Dados II.

## Objetivo

Este dashboard apresenta uma análise histórica das seleções campeãs da Copa do Mundo FIFA, utilizando dados do repositório **jfjelstul/worldcup**, com o objetivo de identificar padrões históricos que possam auxiliar em análises relacionadas à Copa do Mundo de 2026.

## Funcionalidades

- Dashboard interativo em Streamlit
- Filtros por período e seleção campeã
- KPIs da competição
- Ranking das seleções campeãs
- Estatísticas de gols
- Evolução histórica dos campeões
- Distribuição das seleções por continente
- Comparação entre seleções
- Exportação dos dados em CSV
- Galeria de imagens de jogadores

## Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Plotly
